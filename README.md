### Задание 1
Написать учебный лексический анализатор для стандарта языка Паскаль

Исходные данные: 

Файл 1 – программа пользователя на стандарте языка Паскаль (возможно, с ошибками).

Результат: 

Первый этап Файл 2 – коды символов программы из файла 1.

Второй этап написания лексического анализатора включает решение следующих задач: 

• для целых чисел добавить проверку, не выходит ли число за пределы допустимого диапазона;

• для идентификаторов выполнять поиск по таблице ключевых слов.
